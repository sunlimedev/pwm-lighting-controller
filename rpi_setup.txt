Perform the following commands system-wide on a fresh OS:

$ sudo apt update -y
$ sudo apt upgrade -y
$ sudo raspi-config nonint do_i2c 0
$ sudo apt install swig python3-dev build-essential
$ sudo apt install liblgpio-dev

Once finished, restart your device.
Perform the following commands to create a project folder and install the necessary packages inside of a Python Virtual Environment:

$ mkdir project
$ cd project
$ python -m venv venv
/project $ python -m venv venv
/project $ source venv/bin/activate
(venv) /project $ pip install gpiozero
(venv) /project $ pip install python-time
(venv) /project $ pip install thread
(venv) /project $ pip install adafruit-blinka
(venv) /project $ pip install adafruit-circuitpython-pca9685
(venv) /project $ pip install lgpio
(venv) /project $ pip install Flask
